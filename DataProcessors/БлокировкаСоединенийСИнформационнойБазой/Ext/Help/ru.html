Обработка "Блокировка соединений с информационной базой"

 

Обработка «Блокировка соединений с информационной базой» предназначена для блокировки соединений Пользователей с информационной базой. Данная обработка используется Администратором системы при необходимости проведения работ, связанных, например, с настройкой системы. Т.е. для работ, для которых необходимо запретить использование ИБ для всех активных пользователей. Обработка «Блокировка соединений с информационной базой» является стандартной обработкой 1С:Предприятие 8.2.

 

Запрет на использование пользователями информационной базы указывается в форме обработки при помощи реквизитов «Начало действия блокировки» и «Окончание действия блокировки».

За 5 минут до начала действия обработки всем подключенным пользователем выдаются предупреждения о планируемом времени начала и окончания блокировки соединений при помощи текста, указанного в текстовом поле «Сообщение».
