Справочник "Процедуры БП"

 

Справочник «Процедуры бизнес процессов» предназначен только для создания и хранения наименования процедур бизнес-процессов. Сама же процедура бизнес-процесса отражает суть проведения бизнес-процесса. Но для каждой процедуры БП может существовать несколько различных вариантов проведения бизнес-процесса, которые, сохраняя его суть, могут отличаться по последовательности и результатам выполнения отдельных стадий, например, в применении для различных организаций. Произвольная процедура действий является предопределённым элементом.

Справочник «Процедуры БП» имеет одноуровневую структуру.


Форма элемента справочника состоит из:

·       рабочей области с реквизитами справочника;

·       области навигации, которая по ссылке позволяет перейти к связанному справочнику «Результаты процедур бизнес-процессов».
