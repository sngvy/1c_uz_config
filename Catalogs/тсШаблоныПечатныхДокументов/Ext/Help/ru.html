Справочник "Шаблоны печатных форм"

 

Справочник «Шаблоны печатных форм» служит для создания и хранения шаблонов печатных форм документов.

Справочник имеет многоуровневую структуру. 

 

Форма элемента справочника состоит из реквизитов шапки формы и трех закладок: «Реквизиты», «Таблицы» и «Используемые свойства».

Основные реквизиты шапки элемента справочника:

·       На примере документа – наименование документа, для которого осуществляется выбор или создание данного шаблона печатных документов. Выбор документа осуществляется в два этапа:

   o   На первом этапе из формы «Выбор типа данных» выбирается тип журнала документов;

   o   На втором этапе из выбранного журнала документов выбирается сам документ;

·       Путь к шаблону – директория размещения шаблона печатного документа.

 

           Закладка «Реквизиты» предназначена для отражения состава реквизитов шаблона печатных форм в виде табличной части. Особое внимание обратить на графы:

·       Поле – аббревиатура поля автозамены, идентичная аббревиатуре ключевого поля в шаблоне печатного документа с расширением (*.dot), созданного в редакторе Microsoft Word;

·       Тип поля – тип поля автозамены:

   o   статический текст (при выборе данного типа поля содержимое в колонке «Значение» заполняется вручную);

   o   реквизит (выбирается из перечисления); 

·       Значение – значение, которое принимает реквизит в шаблоне печатной формы, заполнение данной графы зависит от значения, заданного в графе «Тип поля»;  

·       Функция – данная графа заполняется при необходимости преобразования значения реквизита текстового шаблона, заданного в графе «Значение»; (выбирается из справочника «Функции текстовых шаблонов»);

·       Пустое значение не пробел - если при формировании печатной формы документа при наличии пустого значения тэга необходимо полностью скрыть данное пустое значение в электронной форме документа, то в данном реквизите устанавливается флажок. Если вместо пустого значения тэга необходимо установить пробел (например, выделенный цветом для контроля параметров автозамены), то флажок не устанавливается.

 

Закладка «Таблицы» предназначена для формирования таблиц в шаблонах печатных форм. Закладка состоит из командной панели и двух табличных частей. В верхней табличной части задается наименование таблиц, а в нижней - графы таблицы. Причем, при позиционировании на какой-либо строке верхней табличной части в нижней табличной части отражаются данные шаблона таблицы выбранного наименования.

Создание нового шаблона печатной формы таблицы производится в верхней табличной части. Особое внимание обратить на графы:

·       Поле таблицы – аббревиатура наименования таблицы, идентичная аббревиатуре наименования таблицы в шаблоне печатного документа с расширением (*.dot), созданного в редакторе Microsoft Word;

·       Функция – наименование алгоритма запроса на языке программирования 1С:Предприятие 8.2 (выбирается из элементов справочника «Функции текстовых шаблонов»);

 

Нижней табличной части закладки особое внимание обратить на графы:

·       Поле – аббревиатура наименования графы таблицы, идентичная аббревиатуре наименования графы таблицы в шаблоне печатного документа с расширением (*.dot) редактора Microsoft Word;

·       Значение - вид результата автозаполнения графы таблицы в шаблоне печатной формы;

·       Функция – наименование алгоритма запроса на языке программирования 1С:Предприятие 8.2 (выбирается из элементов справочника "Функции текстовых шаблонов" при необходимости внесения изменений в представление данных графы «Значение»).

 

Закладка «Используемые свойства» предназначена для формирования списка дополнительных свойств, используемых в данном шаблоне. Закладка состоит из командной панели управления и табличной части, в которую  вручную добавляются дополнительные свойства из справочника «Дополнительные реквизиты и сведения». При заполнении табличной части особое внимание обратить на графы:

·       Группа свойств – группа свойств, к которой принадлежит выбранный элемент справочника «Дополнительные реквизиты и сведения» (заполняется автоматически при заполнении элемента из графы «Свойства»);

·       Порядковый номер – порядок следования (расположения) свойств в форме для внесения дополнительных сведений в соответствии с элементом справочника «Дополнительные реквизиты и сведения»;

·       Заголовок – автозамена наименования данного дополнительного свойства, отражённого в графе «Свойство» данной таблицы в электронной форме печатного документа;

·        Функция – наименование алгоритма запроса на языке программирования 1С:Предприятие 8.2 (выбирается из элементов справочника «Функции текстовых шаблонов» при необходимости внесения изменений в представление данных графы «Значение») или из графы «Заголовок», если для данной строки эта графа заполнена.
