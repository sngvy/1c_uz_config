Справочник "Валюты"

 

Справочник «Валюты» предназначен для хранения списка валют, используемых во всех хозяйственных операциях предприятия, связанных с движением денежных средств в валюте. Справочник «Валюты» имеет одноуровневую структуру. Информация о курсе и кратности валют хранятся в регистре сведений. По умолчанию в таблице указываются значения курсов валют, установленные на рабочую дату, которая отражается в графе таблицы «Курс на _______ г., кратность».


Динамику изменения курса, необходимость автоматической загрузке курса данной валюты при запуске ПК «ПДУ» и другие вспомогательные сведения о валюте можно посмотреть, открыв форму элемента справочника. Переход к просмотру курсов валют осуществляется при помощи ссылки «Курсы валют», находящейся в панели навигации формы, при этом открывается таблица изменения курса валюты. При помощи стандартных клавиш сортировки и отбора, расположенных в командной панели таблицы, можно отсортировать список курсов валют или установить период отображения истории курсов валют.

 

Закладка «Параметры прописи валюты» предназначена для внесения параметров прописи валюты, что необходимо для правильной подстановки наименования валюты прописью в различных электронных или печатных формах, например, в печатных формах документов или при автоинформировании должников при помощи рассылки SMS-сообщений.

 

Кнопка «Подбор из классификатора» командной панели элемента справочника предназначена для ввода в справочник новой валюты из общероссийского классификатора валют.

 

Кнопка «Загрузка курсов» командной панели элемента справочника предназначена для автоматического ввода в справочник данных о курсе валют. В данную форму попадают все валюты из списка валют справочника «Валюты». Выбор валют, для которых будет производиться загрузка курсов, осуществляется установкой флажка напротив наименования нужной валюты. В реквизите «Период загрузки» вводится период, за который будут загружены курсы валют. Далее при помощи кнопки «Загрузить» производится соединение вашего компьютера с сайтом компании «РосБизнесКонсалтинг» (РБК) (www.rbc.ru) и автоматическая загрузка курсов валют за выбранный вами период.
