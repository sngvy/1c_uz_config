Основная форма документа БИТ.Phone Список обзвона

Документ предназначен для автоматизированного обзвона через форму оператора БИТ.Phone. Табличная часть документа содержит список контрагентов с номерами телефонов. Флаг "Обработан" указывает, какие номера списка уже были обработаны при автоматизированном обзвоне. Также для контрагента можно указать контактное лицо. Доступен ускоренный подбор контрагентов и быстрая очистка флагов "Обработан" для всего списка.
