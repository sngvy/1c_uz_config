Обработка "Подбор ДО"

 

В программном комплексе для удобства работы с большими объемами данных в документах конфигурации создана унифицированная система подбора долговых обязательств. Она реализована в специальной обработке «Подбор ДО», вызываемой при помощи кнопки «Подбор» командной панели табличной части форм различных документов, например в форме документа конфигурации «Акт передачи долговых обязательств».

 

***********************************************************

Примечание:      Обработка «Подбор ДО» функционирует только, если в реквизите «Основной режим запуска», когда переключатель установлен в положении «Толстый клиент»!

************************************************************

 

При использовании данной кнопки Пользователю выводится диалоговая форма обработки. Данная диалоговая форма состоит из двух табличных частей. В верхней части обработки «Отбор» осуществляется выбор вида и значения отбора, а в нижней части «Долговые обязательства» – выводится список долговых обязательств, удовлетворяющих условию отбора.

В верхней части обработки рекомендуется указать параметры отбора: вид и значения отбора для элемента или группы элементов, тип сравнения и необходимость использования данного отбора. При вводе нового вида отбора в поле выбора необходимо при помощи формы «Выбор поля» указать тип доступного поля. Выбор вида сравнения осуществляется при помощи стандартных функций и процедур.


 

Далее при нажатии кнопки "Отобрать" командной панели табличной части «Долговые обязательства», Пользователю выводится список обязательств, удовлетворяющих условиям отбора. Т.е. только тех долговых обязательств, которые последовательно удовлетворяют всем условиям отбора, выбранных Пользователем в табличной части «Отбор» и напротив которых установлен соответствующий флажок. Выбор долговых обязательств, удовлетворяющих условиям отбора и необходимых Пользователю, осуществляется в табличной части «Долговые обязательства» формы обработки при помощи установки / снятия флажков в графе «Пометка» напротив выбранных строк.

 

Перенос подобранных долговых обязательств в форму документа с одновременным закрытием формы обработки «Подбор ДО» осуществляется при использовании кнопки «Добавить и закрыть»  в верхней части формы.

 

Для сохранений условий отбора (настроек отбора) с целью дальнейшего уменьшения времени на поиск необходимого типа информации и значений отборов  предусмотрена кнопка командной панели обработки «Сохранить настройки». При использовании данной кнопки Пользователю открывается форма «Настройки подбора», в которой необходимо заполнить только наименование настройки в произвольной форме.  В дальнейшем любой сохранённый вариант отбора возможно загрузить при помощи реквизита «Настройка» шапки формы обработки.

 

При необходимости сгруппировать несколько условий отбора для установления не только вида отбора по умолчанию (автоматически) «И» (последовательно удовлетворяющих условиям отбора), но и отбора вида «Или» (удовлетворяющие хотя бы одному из условий отбора) предназначена кнопка командной панели формы «Сгруппировать условия». Для использования условия «И» / «Или» необходимо сначала воспользоваться кнопкой «Добавить новый элемент, а далее использовать кнопку «Сгруппировать условия». При необходимости соблюдения условия «Или» исправить наименование группы вручную на «Группа Или». Для группирования / разгруппирования условий отбора также используется стандартный функционал «Все действия» панели управления формы.
