Бизнес процесс "Кампании обзвона"

 

Все запущенные бизнес-процессы «Кампании обзвона» отражаются в табличной части рабочей области при активации ссылки «Кампания обзвона» в области навигации. Протекание бизнес-процессов обзвона осуществляется только через ПК «ЦКТ». Создание новых, а также просмотр и редактирование бизнес-процессов осуществляется при помощи стандартных инструментов командной панели.

 

Форма элемента бизнес-процесса состоит из реквизитов шапки, а также табличных частей «Дополнительные операторы» и «Должники». При заполнении шапки формы особое внимание обратить на реквизиты:

·       Приоритет – значение очерёдности запуска данного бизнес-процесса (величина данного реквизита выбирается из перечисления: «Очень низкий», «Низкий», «Средний», «Высокий» и «Самый высокий»); 

·       Повторов – количество повторов на запуск данного бизнес-процесса по телефонам должников при отсутствии с ними состоявшегося диалога (выбирается в интервала от 1 до 9);

·       Вид КИ – вид контактной информации (выбирается из элементов справочника «Дополнительные реквизиты и сведения»);

·       Группа операторов – группа операторов для обзвона группы должников, отобранных в группы по результатам выполнения скоринга (выбирается из элементов справочника «Группы операторов»).

 

В табличной части «Дополнительные операторы» при необходимости увеличения количества операторов для обзвона определённой группы должников по данному бизнес-процессу при помощи стандартных инструментов командной панели могут быть временно добавлены дополнительные операторы (выбираются из элементов справочника "Пользователи").

 

В правой части формы бизнес-процесса «Компании обзвона» расположена таблица, для ввода списка должников, подлежащих обзвону по данному бизнес-процессу. Заполнение таблицы осуществляется при помощи обработки «Подбор ДО», вызываемой при использовании кнопки «Подбор», расположенной в командной строке таблицы.

 

После заполнения формы бизнес-процесса «Кампании обзвона», при помощи кнопки «Стартовать и закрыть» выполняется запуск бизнес-процесса. При этом сервер с программным комплексом лишь сообщает серверу «ЦКТ» о готовности данного бизнес-процесса, а запуск самого бизнес-процесса «Кампания обзвона» осуществляет сервер «ЦКТ».
