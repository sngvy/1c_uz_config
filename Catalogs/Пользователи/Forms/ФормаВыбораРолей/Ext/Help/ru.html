Форма позволяет выбрать или подобрать разрешенные действия (роли).

Список разрешенных действий (ролей) - это список синонимов ролей (как они заданы в конфигураторе), отсортированный по возрастанию.

При подборе ролей в профиль может понадобиться:

1) увидеть имена ролей (как они заданы в конфигураторе);
для этого нужно включить обычно скрытое поле "Имя роли" с помощью команды "Изменить форму";

2) отобрать роли по подсистемам;
для этого нужно включить обычно скрытое поле "Имена подсистем" или "Синонимы подсистем" с помощью команды "Изменить форму",
и далее используя команду "Найти" сделать отбор ролей одинаковых подсистем.
