Документ "Формирование претензионных писем"

 

Данный документ предназначен для фиксирования факта создания претензионных писем контрагентам с использованием шаблонов печатных форм документов.

 

Форма документа состоит из командной панели формы, реквизитов шапки документа и табличной части, содержащей список объектов, для которых данным документом будут сформированы претензионные письма.

 

При заполнении шапки формы документа особое внимание обратить на правильность заполнения реквизитов:

·       Шаблон – наименование шаблону для формирования претензионных писем (выбирается из элементов справочника «Печатные формы»);

·       Автор – наименование Пользователя, создавшего данный документ.

 

Далее Пользователь должен заполнить табличную часть документа. Заполнение табличной части производится при помощи стандартных функций управления командной панели. Кнопка «Печать УЭД» командной панели формы предназначена для распечатки сформированных данным документом претензионных писем.
