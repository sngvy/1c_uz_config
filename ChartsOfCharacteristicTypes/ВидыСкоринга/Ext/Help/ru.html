Справочник "Виды скоринга"

 

Справочник «Виды скоринга» предназначен для хранения информации о видах автоматической группировки данных, используемых в программном комплексе для присвоения объектам определенных категорий.

Справочник имеет одноуровневую структуру.

 

Важно!!!:       Создание вида скоринга необходимо осуществлять ТОЛЬКО в режиме работы с информационной базой «Толстый клиент», устанавливаемой в параметре «Основной режим запуска»! Данное ограничение связано со структурой работы платформы 1С: Предприятие 8.2 и отражается на работоспособности Конструктора запросов!

 

Форма элемента справочника состоит из рабочей области, содержащей реквизиты шапки и табличными частями, с данными о категориях для данного вида скоринга, и области навигации, которая позволяет перейти к справочнику «Категории».

 

Справочник «Виды скоринга» состоит из двух табличных частей:

·       «Виды скоринга», содержащей виды группировки данных;

·       «Категории скоринга», содержащей все категории (условия группировки) скоринга, выделенного в верхней таблице.

 

При заполнении шапки формы элемента справочника особое внимание обратить на реквизиты:

·       Объект скоринга – наименование типа данных для автоматической группировки данных (выбирается из перечисления):

   ·       Расписание – реквизит, определяющий периодичность проведения данного вида скоринга;

·       Используется – в данном реквизите устанавливается флажок в случае необходимости использования данного элемента справочника при проведении регламентных заданий в программном комплексе.

 

В табличной части «Категории» элемента справочника содержится информация о категориях данного вида скоринга. В графе "Условие попадания в категорию " описаны условия, при выполнении которых объект скоринга попадает в данную категорию.

 

Кнопка «Запрос» командной панели справочника используется Администратором для создания новых категорий скоринга. А форма «Создать на основании», расположенная в командной панели справочника используется Администратором для создания одного вида скоринга на основании другого вида скоринга.
