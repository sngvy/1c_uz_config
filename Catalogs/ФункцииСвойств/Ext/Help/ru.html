Справочник "Функции свойств УЭД"

 

Справочник «Функции свойств УЭД» предназначен для создания и хранения функций, определяющих шаблоны печатных форм при формировании печатных форм документов. Элементы данного справочника используются при заполнении закладки «Используемые свойства» справочника «Шаблоны печатных форм».

Справочник «Функции свойств УЭД» имеет одноуровневую структуру.

 

Форма элемента справочника «Функции свойств УЭД» состоит из командной панели формы, реквизитов справочника, панели «Функция» и табличной части «Объект Вх».

В панель «Функция» записывается текст функции на языке программирования 1С:Предприятие 8.2.  

 

Табличная часть «ОбъектВх» содержит следующие графы:

·       № – порядковый номер свойства;

·       Свойство – наименование свойства (выбирается из элементов справочника «Дополнительные реквизиты и сведения»);

·       Значение – значение свойства; заполняется в два этапа:

o    На первом этапе выбирается тип значения из открывающейся формы;

o    На втором этапе выбирается само значение свойства из соответствующих справочников и форм.
