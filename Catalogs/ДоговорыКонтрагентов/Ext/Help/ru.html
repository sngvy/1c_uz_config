Справочник "Договоры контрагентов"

 

Справочник «Договоры контрагентов» предназначен для хранения договоров контрагентов, связанных с истребованием задолженности. Применяется для формирования элементов справочника «Услуги по договору».

Новые элементы в данный справочник загружаются из «Excel»-таблиц при помощи подсистемы «Ввод-вывод данных или заносятся вручную. Редактирование и ввод новых элементов в справочник осуществляется при помощи стандартных функций управления командной панели, при этом Пользователю открывается форма элемента справочника «Договоры контрагентов».

 

Форма элемента справочника состоит из рабочей области с реквизитами, содержащими данные о договоре с контрагентом, и области навигации, которая позволяет перейти к справочникам, связанным с данным договором с контрагентом: «Задолженность по объектам остатки», «Мероприятия», «Ответственные сотрудники», «Услуги по договору» и т.д.
