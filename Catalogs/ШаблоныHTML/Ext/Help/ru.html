Справочник "Шаблоны HTML"

 

Справочник «Шаблоны HTML» используется для создания текстовых шаблонов для отправки сообщений (извещений) по электронной почте (e-mail сообщения) при автоинформировании контрагента.

 

Форма элемента справочника состоит из реквизитов шапки формы, двух закладок «Основная» и «Табличный документ» и поле просмотра шаблона «HTML документ».  

Закладки «Основная» и «Табличный документ» предназначены для заполнения кода HTML. При этом закладка «Основная» предназначена для заполнения текста шаблона, а закладка «Табличный документ» – для загрузки кода шаблона, используя таблицы шаблона.

 

В нижней части формы элемента справочника расположено поле просмотра шаблона «HTML документ», в котором отображается готовый HTML документ, в том виде, в котором он будет выглядеть в интернет-браузере. Причем, теги, выделенные знаками «@» в обоих сторон, в тексте письма будут заменены значениями соответствующих реквизитов.

 

В программном комплексе предусмотрена возможность вставки рисунка и фона в HTML шаблон, для этого при написании кода шаблона делается ссылка на файл с рисунком.
