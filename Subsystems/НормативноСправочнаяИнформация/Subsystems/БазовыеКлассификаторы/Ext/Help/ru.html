Формирование и поддержка классификатора стран мира

Список стран мира можно открыть из раздела "Нормативно-справочная информация".

При добавлении новой записи о стране мира нужно указать ее название и код.

Поддерживается возможность множественного добавления элементов из ОКСМ (общероссийского классификатора стран мира).

Формирование и поддержка классификатора валют

Работа со списком валют

Перечень валют можно посмотреть в списке, который открывается из раздела "Нормативно-справочная информация".

Добавить новые записи в список валют можно, использовав общероссийский классификатор валют (ОКВ). Его можно открыть из списка валют. В форме ОКВ выбирается валюта, которую нужно добавить. После этого автоматически открывается форма нового элемента, в которой такие данные как "код валюты", "наименование", "полное наименование", "признак загрузки курса с сайта РБК" уже будут заполнены информацией с ОКВ. Остальные данные вводятся вручную пользователем. Из окна ОКВ можно добавить подряд несколько записей о валюте.

Создание новой записи о валюте

При добавлении новой записи обязательно указание названия валюты. Дополнительно можно указать полное наименование.

Для того чтобы в формах печати правильно формировалось наименование валюты прописью, необходимо задать параметры прописи валюты. Для валюты задаются параметры прописи целой и дробной части валюты. В том случае, если дробная часть в печатных формах отображается числом, то можно установить нужное количество разрядов при выводе печатных форм документов

Загрузка курсов валют

Перечень валют, для которых выполняется загрузка, можно выбрать вручную, можно подобрать автоматически по включенному признаку "Валюта загружается с сайта РБК". Далее нужно установить период, за который будет выполнена загрузка курсов, и выполнить загрузку.

Чтобы посмотреть историю курсов валют нужно открыть форму элемента для выбранной валюты и перейти к окну просмотра курсов.

Формирование и поддержка классификатора банков

Поддерживается возможность загрузки банков из классификатора банков РФ с диска информационно-технологического сопровождения.

Поддерживается возможность загрузки банков из классификатора банков РФ с сайта "Росбизнесконсалтинг".

Работа с банком

Создать новую запись о банке можно вручную. Заполняется информация:
- Наименование банка
- БИК банка
- Корреспондентский счет банка
- Город и адрес банка
- Группа банков

В форме элемента выводятся следующие данные:
- Наименование банка
- БИК банка
- Корреспондентский счет банка
- Город и адрес банка
- Группа банков

В форме группы банков выводятся следующие данные:
- Наименование группы
- Код региона

Работа со списком банков

Список банков можно открыть из раздела "Нормативно-справочная информация".

В форме списка выбора банков выводится следующая информация:
- Наименование банка
- БИК банка
- Корреспондентский счет банка
- Город и адрес банка

Предусмотрена возможность добавления банков из классификатора банков. Классификатор банков встроен в конфигурацию и представляет собой mxl-таблицу со списком всех возможных банков.

Чтобы найти в классификаторе нужный вам банк можно воспользоваться фильтрами для поиска. Поиск подходящих банков можно вести по "БИК", "Корреспондирующему счету", "Наименованию", "Городу". В соответствующих полях нужно указать искомое значение (полностью или частично). После этого можно автоматически подобрать банки, отвечающие указанным условиям поиска. Несколько условий объединяются по "И". В отобранном списке можно выбрать один или несколько банков. Эти банки будут добавлены в список банков системы.

Настройка календарей

В системе можно хранить список записей о календарях на каждый год. Список календарей открывается из раздела "Нормативно-справочная информация".

В список календарей можно ввести запись о новом календаре. В новом календаре вводится его название и определяется список рабочих и выходных дней. Для каждого дня нового календаря нужно установить признак является ли день рабочим или выходным.

По нажатию кнопки "Все действия"- "Заполнить календарь" выполняется автоматическое заполнение календаря пятидневкой без учета праздничных дней за год, указанный на форме. Далее пользователю необходимо будет вручную настроить календарь указав праздничные дни.

Предусмотрена возможность заполнения календаря по аналогии с ранее введенным календарем ("Все действия" - "Заполнить по календарю").
