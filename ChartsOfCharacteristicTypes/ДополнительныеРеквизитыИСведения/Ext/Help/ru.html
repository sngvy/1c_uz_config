Справочник "Дополнительные реквизиты и сведения"

 

Справочник «Дополнительные реквизиты и сведения» предназначен для внесения и хранения дополнительной информации, используемой при заполнении элементов различных справочников (дополнительных реквизитов и дополнительных свойств объектов).

Справочник имеет многоуровневую структуру.

Добавление групп в справочник и элементов в каждую из групп справочника осуществляется при помощи стандартных функций управления командной панели. При добавлении и редактировании групп справочника Пользователю открывается форма «Дополнительные сведения», в которой помимо стандартных реквизитов форм ("Код", "Наименование" и т.д.), указывается "Порядковый номер"  – порядок следования (расположения) свойств в форме для внесения дополнительных сведений.

 

Первый уровень справочника «Дополнительные реквизиты и сведения» составляют группы, созданные для справочников ПДУ, к которым добавляются дополнительные реквизиты и сведения.

 

Второй уровень справочника «Дополнительные реквизиты и сведения» содержит по две группы для каждого элемента первого уровня: «Дополнительные реквизиты» и «Дополнительные сведения», предназначенные для элементов справочников.

В первой группе второго уровня («Дополнительные реквизиты») содержатся элементы справочника, которые отображаются в панели «Дополнительные реквизиты» форм элементов справочников. После сохранения элементов справочника «Дополнительные реквизиты и сведения» данные дополнительные реквизиты отражаются в форме элементов справочников в панели «Дополнительные реквизиты».

 Элементы второй группы второго уровня («Дополнительные сведения») отображаются при открытии дополнительного окна форм справочников ПДУ при помощи кнопки «Доп. сведения», расположенной в командной строке справочников. В эту группу может быть внесено неограниченное количество групп дополнительных сведений для каждой группы более высокого уровня. Эти дополнительные сведения выводятся в отдельном окне справочника при нажатии кнопки «Доп. сведения», расположенной в командной строке элемента справочника.

 

Важно!!!:          Внесение любых дополнительных сведений необходимо осуществлять ТОЛЬКО в подгруппы группы «Дополнительные сведения»!!!

 

При создании элемента справочника в каждой их групп второго и последующих уровней Пользователю для заполнения открывается форма «Принимаемые значения». Вид формы отличается для различных типов значения дополнительных реквизитов и свойств: «Строка», «Число», «Дата», «Значение свойств объектов» и т.д.

 

I. Тип значения дополнительных реквизитов простейших типов

Форма «Принимаемые значения» для значения дополнительных реквизитов простейших типов «Строка», «Число» и «Дата» состоит из реквизитов формы, включающих как тип элемента, так и тип значения данного элемента:  

·       Порядковый номер – порядок следования реквизита или дополнительного свойства в соответствующей форме;

    ·       Справочник владелец – справочник конфигурации, для которого создается данный элемент;

·       Тип элемента – тип элемента справочника («Дополнительные реквизиты» или «Дополнительные сведения»);

·       Группа свойств – группа элементов справочника, в которую входит данный элемент;

·       Тип значения – тип значения дополнительных реквизитов или сведений (выбирается из перечисления);

·       Вид строки – вид вносимой строки (доступен для заполнения только для типа значения «Строка» из перечисления):

·       Вид КИ – вид контактной информации (реквизит доступен для заполнения только для типа значения «Строка» и выбирается из справочника «Виды контактной информации»).

 

II. Тип значения дополнительных реквизитов «Значение свойств объектов»

Форма «Принимаемые значения» для типов значения дополнительных реквизитов «Значение свойств объектов» отличается от описанной выше формы только наличием табличной части «Принимаемые значения». Содержание реквизитов и их заполнение аналогично форме для типов значения дополнительных реквизитов простейших типов. В табличную часть «Принимаемые значения» вносятся возможные принимаемые значения данного дополнительного реквизита или свойства.

При заполнении форм типов объектов (типов контрагентов, типов долговых обязательств и т.д.) Пользователю предоставлена возможность самостоятельного создания набора дополнительных свойств (сведений) и реквизитов для различных типов описываемых объектов. Например, для типа контрагентов - физические лица в качестве дополнительных сведений можно указывать адрес прописки, а для юридических лиц - фактический и юридический адрес.
