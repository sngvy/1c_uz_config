Документ "Передача на выполнение"

 

Документ «Передача на выполнение» предназначен для регистрации факта передачи на выполнение определенного бизнес-процесса (или процедуры БП) в другую организацию.

 

Форма документа «Передача на выполнение» состоит из командной панели формы с элементами управления формой документа, реквизитов шапки формы, двух панелей: «Передает» и «Принимает», а также табличной части «Долговые обязательства».

 

Панель «Передает» содержит следующие реквизиты:

·       Организация – организация, передающая объекты для запуска бизнес-процессов (заполняется автоматически в соответствии с принадлежностью к определенной организации Пользователя, создающего данный документ);

·       Подразделение – подразделение организации (выбирается из перечисления, сформированного из элементов справочника «Подразделения»).

 

Панель «Принимает» содержит следующие реквизиты:

·       Организация – организация, принимающая объекты для запуска бизнес-процессов (выбирается из перечисления, сформированного из элементов справочника «Организации»);

·       Процедура – процедура бизнес-процесса, для выполнения которой объекты передаются в другую организацию (выбирается из элементов справочника «Процедуры бизнес-процессов»).

 

В табличную часть «Долговые обязательства» при помощи стандартных функций управления командной панели или обработки «Подбор ДО» вносятся однотипные долговые обязательства, передаваемые в другую организацию.
