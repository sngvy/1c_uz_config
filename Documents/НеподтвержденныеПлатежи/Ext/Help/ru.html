Документ "Неподтверждённые платежи"

 

Документ «Неподтвержденные платежи» предназначен для регистрации неподтвержденных фактов оплаты платежей по долговым обязательствам со слов контрагентов или контактных лиц контрагентов. Документы «Неподтвержденные платежи» как правило создаются автоматически при выполнении мероприятий, включающих в себя контакт с контрагентом, но могут и быть созданы Пользователем вручную.


Форма документа состоит из реквизитов шапки формы и табличной части. Реквизиты шапки формы документа заполняются автоматически на основании данных документа «Мероприятие».

Шапка формы документа содержит следующие реквизиты:

·       Номер – номер документа (автоматически генерируется системой);

·       Дата – дата создания документа (может быть изменена Пользователем вручную с текущей даты на произвольную);

·       Объект – наименование объекта, для которого регистрируется данный обещанный, но неподтверждённый платёж. При формировании документа вручную объект выбирается в два этапа:

o   на первом этапе из открывающейся формы «Выбор типа данных» выбирается тип объекта;

o   на втором этапе - сам объект (из элементов соответствующего справочника).

·       Автор – наименование Пользователя, создавшего данный документ.

 

Табличная часть документа «Неподтвержденные платежи» содержит список неподтвержденных платежей по выбранному выше объекту, которые фиксируются данным документом.
