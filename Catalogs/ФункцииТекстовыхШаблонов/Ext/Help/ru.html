Справочник "Функции текстовых шаблонов"

 

Справочник «Функции текстовых шаблонов» предназначен для создания текстовых шаблонов автоинформирования должников.

Справочник имеет одноуровневую структуру.

Форма элемента справочника «Функции текстовых шаблонов», которая состоит из реквизитов шапки формы и четырех панелей: «Входные данные», «Возвращаемые значения», «Текст функции» и «Выходные данные».

 

Значения реквизитов панели «Входные данные» используется для тестирования функции текстовых шаблонов, а в панель «Текст функции» записывается алгоритм функции на языке программирования 1С:Предприятие 8.2.

 

Панель «Возвращаемое значение» предназначена выбора типа возвращаемого значение функции. Реквизит «Значение» панели «Выходные данные» заполняется автоматически при нажатии кнопки «Тест» после заполнения всех необходимых реквизитов элемента справочника.
