Справочник "Пользователи"

 

Справочник «Пользователи» предназначен для хранения информации о пользователях данного программного комплекса.

Справочник «Пользователи» имеет многоуровневую структуру.

Просмотр элементов справочника осуществляется при помощи стандартных функций управления командной панели.

Форма элемента справочника состоит из реквизитов шапки формы и трех закладок: «Дополнительные реквизиты», «Настройки программы» и «Настройки оператора».

 

При заполнении шапки формы особое внимание необходимо обратить на следующие реквизиты:

·       Руководитель – в данном реквизите устанавливается флажок, в случае если Пользователь является руководителем подразделения организации; 

·       Уволенный сотрудник – при установке флажка в данном реквизите пароль Пользователя для входа в систему и его логин перестаёт быть актуальным. Наименование Пользователя также пропадает с Рабочего стола из структуры организации.

 

Закладка «Дополнительные реквизиты» используется для отражения в формах дополнительных реквизитов данного Пользователя, например, его контактной информации.

 

Закладка «Настройки программы» предназначена для отражения отдельных функций программы по работе с данным Пользователем. Наличие и количество необходимых настроек определяется элементами справочника «Настройки Пользователей».

  

Закладка «Настройки оператора» предназначена для настроек прав Пользователя, например, как оператора Контакт-центра или сотрудника отдела техподдержки. Наличие и количество групп операторов определяется элементами справочника «Группы операторов».
