Справочник "Функции комментариев"

 

Справочник «Функции комментариев» предназначен для создания и хранения функций, определяющих шаблон комментариев для формирования документа «Мероприятие». Элементы данного справочника используются при заполнении поля «Комментарий» в карточке мероприятий.

Справочник «Функции комментариев» имеет одноуровневую структуру.

 

При заполнении шапки элемента справочника особое внимание обратить на реквизит "Значение" - результат срабатывания функции при проверке правильности её функционирования. 

В реквизите «МероприятиеТекущее» панели "Входные данные" выбирается текущее мероприятие, для которого формируется шаблон комментариев, а в панели "Запрос" Администратором системы на языке программирования 1С:Предприятие 8.2 составляет текст функции при помощи конструктора запроса и конструктора формата.
