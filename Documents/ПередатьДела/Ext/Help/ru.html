Документ "Передать объекты"

 

Документ «Передать объекты» предназначен для регистрации факта передачи объектов для работы с ними от одного сотрудника к другим сотрудникам внутри одной организации. Форма документа состоит из реквизитов шапки формы и трех табличных частей.

 

В табличной части «Передаваемые типы» указываются все роли сотрудника, передающего объекты (заполняется из элементов справочника «Типы сотрудников»).

 

В табличную часть «Сотрудники принимающие» вносятся сотрудники (или сотрудник), которым передаются объекты (заполняется из элементов справочника «Пользователи»).

 

Табличная часть «Объекты» предназначена для подбора объектов, которые передаются данным документом от одного сотрудника другому. Для подбора объектов системы используется или ручной способ, или автоматический (при помощи кнопки «Подбор» командной панели табличной части «Объекты», которая вызывает обработку "Подбор долговых обязательств"). Для распределения выбранных объектов между сотрудниками, указанными в табличной части «Сотрудники принимающие», используется кнопка «Распределить» командной панели табличной части «Объекты».

Если в табличной части «Сотрудники принимающие» указано несколько сотрудников, которым предполагается передача дел (объектов), то программный комплекс после использования кнопки «Распределить»:

·       оценивает количество текущих дел каждого выбранного сотрудника (объектов, отражаемых на Рабочем столе сотрудника), а также его роли (права);

·       предлагает распределение передаваемых объектов (а также, мероприятий, связанных с ними) по сотрудникам до равномерного их количества у каждого из сотрудников;

·       автоматически дозаполняет недостающие графы табличной части «Объекты».

 

После автоматического распределения объектов, предусмотрена возможность ручной корректировки распределения объектов при помощи стандартных функций управления командной панели.

 

При проведении документа все объекты, выбранные в табличной части, исчезают с Рабочего стола сотрудника, указанного в поле «Сотрудник передающий» и добавляются в Рабочие столы сотрудников, указанных в табличной части «Сотрудники принимающие».
