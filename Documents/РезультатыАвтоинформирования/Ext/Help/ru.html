Документ "Результаты автоинформирования"

 

Документ «Результаты автоинформирования» предназначен для автоматического фиксирования результата выполнения заданий по автоинформированию должников.

 

Документ состоит из шапки документа, содержащей номер и дату документа, и табличной части. В табличной части фиксируются результаты выполнения заданий по автоинформированию:

·       N – номер строки табличной части документа (каждая из строк таблицы соответствует одному заданию на автоинформирование из списка заданий);

·       УИД – уникальный идентификатор задачи, который присваивается каждой задаче автоматически;

·       Результат – результат выполнения задания по автоинформированию;

·       Дата – дата выполнения задания по автоинформированию.
