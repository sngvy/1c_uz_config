Справочник "Результаты мероприятий"

 

Каждый тип мероприятия из справочника «Типы мероприятий» должен иметь набор вариантов завершения выполнения данного типа мероприятий, т.е. результатов мероприятий, которые хранятся в данном справочнике. Справочник «Результаты мероприятий» является подчиненным справочнику «Типы мероприятий».

Справочник «Результаты мероприятий» имеет одноуровневую структуру.

Форма элемента справочника «Результаты мероприятий» состоит из:

·       реквизитов шапки формы элемента;

·       панели «Обязательные поля»,

·       табличной части, содержащей дополнительные реквизиты результатов мероприятий.

 

При заполении шапки элемента справочника необходимо указать значения следующих реквизитов:

·       Наименование – наименование элемента справочника, соответствующее одному из результатов мероприятия данного типа;

·       Тип мероприятия – тип проводимого мероприятия (выбирается из элементов справочника «Типы мероприятий»). 

 

Реквизиты панели «Обязательные поля»:

·       Указать контактное лицо – в данном реквизите устанавливается флажок в случае, если в карточке мероприятий при выборе результата мероприятия, соответствующего данному элементу справочника, необходимо указать контактное лицо;

·       Прикрепить файл – в данном реквизите устанавливается флажок в случае, если для фиксации результата выполнения мероприятия к нему необходимо прикрепить соответствующий файл;

·       Заполнить комментарий – в данном реквизите устанавливается флажок, если согласно принятому порядку работы организации в карточке мероприятия (при выполнении бизнес-процесса) необходимо заполнить поле «Комментарий».

 

В табличной части элемента справочника «Результаты мероприятий» содержится список дополнительных реквизитов результатов мероприятий, которые при выборе данного элемента справочника отражаются в форме (карточке) документа «Мероприятия». Если в графе «Обязательное для заполнения» табличной части элемента справочника для какого-либо дополнительного реквизита установлен флажок, то выполнение мероприятия с результатом, соответствующим выбранному элементу справочника невозможно без обязательного заполнения значения этого дополнительного реквизита.

В реквизите «Функция для дополнительных реквизитов» можно выбрать элемент справочника «Функции дополнительных реквизитов», определяющий функцию после выполнения мероприятия. А в реквизите «Шаблон комментария» - значение, определяющее шаблон комментариев, согласно которому будет автоматически заполнен комментарий в карточке мероприятий (выбирается из элементов справочника «Функции комментариев»).
