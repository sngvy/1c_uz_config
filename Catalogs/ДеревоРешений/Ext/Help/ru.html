Справочник "Сценарии диалога"

 

Справочник «Сценарии диалога» предназначен для хранения различных возможных вариантов диалога Пользователя с контрагентом. Элементы справочника «Сценарии диалога» используются для ведения и регистрации диалога в форме документа «Мероприятия», например, в подсистеме «Рабочий стол».

Справочник имеет одноуровневую структуру.

 

Форма элемента справочника «Сценарий диалога» состоит из реквизитов шапки и поля графической схемы, содержащего блок схему возможных вариантов диалога Пользователя с контрагентом или его контактными лицами.

При заполнении шапки справочника особое внимание обратить на реквизит "Тип мероприятия" – вид проводимого мероприятия, к которому относится данный элемент справочника «Сценарий диалога», (выбирается из справочника «Типы мероприятий»).

 

Блок-схема возможных вариантов диалога содержит элементы, окрашенные разными цветами:

·       элементы, окрашенные голубым цветом, представляют собой альтернативные конечные результаты диалога. Для данных элементов обязательно заполнение реквизита «Результат мероприятия» в соответствии с элементами справочника «Результаты мероприятия»;

·       элементы, окрашенные желтым цветом, представляют собой узловые элементы сценария. Для данных элементов обычно рекомендуется заполнение реквизита «Выбор контактного лица»;

·       элементы, окрашенные красным цветом, представляют собой текущие узловые элементы сценария.
