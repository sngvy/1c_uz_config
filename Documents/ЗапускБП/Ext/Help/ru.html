Документ "Запуск бизнес процессов"

 

Документ «Запуск бизнес-процессов» предназначен для одновременного запуска бизнес-процессов для нескольких долговых обязательств одного типа. 

 

Форма документа состоит из командной панели формы с элементами управления формой документа, реквизитов шапки формы, панели «Точка старта БП» и табличной части, содержащей данные об объектах для которых будут запускаться бизнес-процессы.


При заполнении шапке формы документа особое внимание обратить на следующие реквизиты:

·       Процедура – наименование группы операций (действий) бизнес-процессов (выбирается из элементов справочника «Процедуры Бизнес-процессов»);

·       Схема – последовательность выполнения отдельных блоков (точек БП) каждого бизнес-процесса, который будет запущен данным документом (выбирается из элементов справочника «Карты маршрутов»). 

 

Выбор стадии запуска бизнес процесса производится в форме «Форма точки старта», открывающейся при помощи кнопки «Выбрать стадию». Выбор осуществляется путем позиционирования на нужном элементе блок-схемы бизнес-процесса и нажатием кнопки «Выбрать», расположенной в командной панели формы «Форма точки старта».

 

В панели «Точка старта БП», содержащей данные о точке начала работы по данному бизнес-процессу, содержатся следующие реквизиты:

·       ТССтартовать с выбранной стадии – при установке флажка в данном реквизите старт бизнес-процесса будет осуществляться со стадии, выбранной при помощи кнопки «Выбрать стадию», расположенной в данной панели;

·        ТСПуть – при установке флажка в данном реквизите старт бизнес-процесса будет осуществляться с выбранной ветки выбранной стадии карты бизнес-процесса (заполняется автоматически при выборе стадии запуска бизнес-процесса, но может быть откорректировано вручную);

·       ТСМероприятие – мероприятие, с которого будет осуществляться старт бизнес-процесса (заполняется автоматически при выборе стадии запуска бизнес-процесса).

  

В табличную часть документа при помощи стандартных функций управления командной панели или обработки «Подбор ДО» (вносятся однотипные долговые обязательства, для которых данным документом будут запущены бизнес-процессы.
