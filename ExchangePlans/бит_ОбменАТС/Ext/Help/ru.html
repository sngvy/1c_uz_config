План обмена с БИТ.АТС

Синхронизация контактной информации и внутренних номеров менеджеров с БИТ.АТС.

Реквизиты

Идентификатор базы

Идентификатор базы – реквизит служит для выгрузки информации из нескольких баз 1С на одну АТС, в этом случае в каждой базе 1С должен быть свой идентификатор в узле обмена с этой АТС. Если база 1С одна, реквизит «Идентификатор базы» можно не заполнять и оставить 0.

Хост БД

Хост БД – имя или IP-адрес АТС.

Наименование БД

Наименование БД – название MySQL базы АТС.

Пользователь БД

Пользователь БД – имя пользователя для подключения к MySQL базе АТС.

Пароль БД

Пароль БД – пароль пользователя для подключения к MySQL базе АТС.
