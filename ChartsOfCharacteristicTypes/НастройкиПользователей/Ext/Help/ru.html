Справочник "Настройки пользователей"

 

Справочник «Настройки пользователей» предназначен для отражения необходимых настроек функционала программного комплекса при работе данного Пользователя. Справочник имеет многоуровневую структуру. Два элемента справочника являются предопределенными. Данные элементы справочника используются в настройке функционала программы по работе с отдельными Пользователем

 

При заполнении формы элемента справочника особое внимание обратить на правильность выбора значения реквизита "Тип значения"  из соответствующего  перечисления.
