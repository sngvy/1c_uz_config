Справочник "Исполнительные документы"

 

Справочник «Исполнительные документы» предназначен для хранения информации об исполнительных документах, созданных в процессе востребования долга с должника на стадии исполнительного производства. Справочник имеет одноуровневую структуру.  


Форма элемента справочника состоит из:

·       командной панели формы элемента справочника;

·       рабочей области с реквизитами, содержащими данные о должнике, долговом обязательстве и исполнительном документе по востребованию задолженности по данному долговому обязательству;

·     области навигации, которая позволяет перейти к справочникам, связанным с данным долговым обязательством: «Дополнительные реквизиты мероприятий», «Задолженность по объектам остатки», «Мероприятия» и т.д.

 

Часть реквизитов рабочей области справочника заполняется автоматически из подсистемы «Ввод / вывод информации», другие заполняются вручную. В рабочей области элемента справочника содержатся следующие основные реквизиты:

·       Служба судебных приставов – служба судебных приставов, которой будет передан в работу данный исполнительный документ (выбирается из элементов справочника «Судебные приставы»;

·       Долговое обязательство – наименование долгового обязательства, к которому относится данный исполнительный документ (выбирается из элементов справочника «Долговые обязательства»);

·       Кредиторы по ДО – наименование организации-кредитора для данного долгового обязательства (заполняется автоматически при заполнении реквизита «Долговое обязательство»);

·       Тип ИД – тип исполнительного документа (выбирается из перечисления, сформированного из элементов справочника «Типы исполнительных документов»):

   ·       Дата принятия судебного акта – дата принятия судебного акта в судебном участке;

·       Дата вступления в законную силу – дата вступления данного исполнительного документа в законную силу.
