Справочник "Долговые обязательства"

 

Справочник «Долговые обязательства» предназначен для хранения информации о договорах по предоставлению услуг, оказанных контрагенту, по оплате которых имеются задолженность перед организацией. Для данного программного комплекса такие оказанные услуги определяются как «Долговые обязательства».

 

Форма элемента справочника состоит из:

·       командной панели формы с элементами управления формой элемента справочника;

·       рабочей области с реквизитами шапки, панели «Дополнительные реквизиты» и табличной части, содержащей данные о событиях и мероприятиях, направленных на востребование задолженности от должника. Часть реквизитов шапки рабочей области справочника заполняется автоматически из подсистемы «Ввод / вывод данных», а другие заполняются вручную;

·      области навигации, которая позволяет перейти к формам программного комплекса, непосредственно связанным с данным долговым обязательством: «Ответственные сотрудники», «Бизнес процессы», «Мероприятия»  и т.д.

 

В программном комплексе предусмотрена возможность учёта исковых требований, как по отдельным оказанным контрагенту услугам (по собственному долговому обязательству), так и по сумме оказанных услуг без их расшифровки в рамках одного долгового обязательства.


В панели «Дополнительные реквизиты» рабочей области элемента справочника размещаются дополнительные реквизиты, характеризующие долговое обязательство.  

В информационном поле «Текущая задолженность» рабочей области элемента справочника отражается текущая непогашенная задолженность должника перед организацией с указанием валюты суммы.

В табличной части рабочей области элемента справочника отражаются динамическая информация о мероприятиях, направленных на востребование задолженности от должника (см. документ «Мероприятие»).

 

При помощи кнопки «Создать на основании», расположенной в командной панели элемента рабочей области справочника «Долговые обязательства», можно непосредственно из формы элемента справочника «Долговые обязательства» создать документ «Мероприятие», после заполнения и проведения которого в системе создается новое мероприятие, направленное на истребование долга с должника.

 При помощи кнопки «Дополнительные сведения», расположенной в командной панели элемента справочника «Долговые обязательства», Пользователь имеет возможность внесения в элемент справочника дополнительные сведения, характеризующих данное долговое обязательство.

 Кнопка «Исходящий звонок» командной панели элемента рабочей области элемента справочника предназначена для запуска бизнес-процесса «Звонок исходящий» со сценарием диалога.

Кнопка «Печать УЭД» командной панели элемента рабочей области элемента справочника предназначена для запуска элемента справочника «Печатные формы» подсистемы "Управления электронной документацией" (УЭД), предназначенного для автоматической печати необходимых печатных форм документов, связанных с данным долговым обязательством.
