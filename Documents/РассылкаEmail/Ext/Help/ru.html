Документ "Задания для автоинформирования"

 

Документ «Задания для автоинформирования» предназначен для  формирования заданий по автоинформированию контрагентов.

 

Документ «Задания для автоинформирования» состоит из реквизитов шапки документа и табличной части, в которую вносится список заданий на автоинформирование, запускаемых на выполнение данным документом при помощи сервера «ЦКТ».

 

При заполнении шапки документа особое внимание обратить на значения реквизитов:

·       Действие – тип информирования (выбирается из перечисления: голосовое сообщение, SMS-оповещение, e-mail рассылка);

·       Вид КИ – вид контактной информации (выбирается из справочника «Дополнительные реквизиты и сведения»). Вид контактной информации в значительной степени зависит от выбранного значения реквизита «Действие». Так для выбранных действий «SMS оповещение» и «Голосовое сообщение» целесообразней выбрать номер мобильного телефона контрагента, а для действия «E-mail рассылка» – адрес его электронной почты;

·       Шаблон – шаблон формирования формы на автоинформирования (выбирается из элементов справочника «Шаблоны текста для автоинформирования»).

После проведения документа «Задания для автоинформирования» все задания, входящие в данный документ, передаются в очередь на исполнение в сервер «ЦКТ».
