Справочник "Услуги по договору"

 

Справочник «Услуги по договору» предназначен для хранения информации об услугах, которые могут быть оказаны контрагенту в рамках заключенных договоров. Учёт данных услуг может осуществляться, как по отдельным оказанным контрагенту услугам (по собственному долговому обязательству), так и по сумме оказанных услуг без их расшифровки в рамках одного долгового обязательства.


Форма элемента справочника состоит из:

·       командной панели формы с элементами управления формой элемента справочника;

·       рабочей области с реквизитами, содержащими данные об услуге и договоре с контрагентом, в рамках которого данная услуга оказывается;

·       области навигации, которая позволяет перейти к справочникам, связанным с данной услугой по договору: «Дополнительные реквизиты мероприятий», «Задолженность по объектам остатки», «Мероприятия», «Ответственные сотрудники», «Прикрепляемые файлы».
