Справочник "Регламентные действия по категориям"

 

Справочник «Регламентные действия по категориям» предназначен для хранения информации о регламентных действиях, проводимых для отдельных категорий объектов скоринга. Справочник имеет одноуровневую структуру.  


Форма элемента справочника состоит из реквизитов шапки и табличной части, содержащей данные о регламентных действиях по проведению скоринга по выбранным категориям. При заполнении шапки элемента справочника особое внимание обратить на заполнение реквизитов:

·       Расписание – реквизит, определяющий периодичность проведения регламентного действия;

·       Используется – в данном реквизите устанавливается флажок в случае использования данного элемента справочника при проведении регламентных операций в программном комплексе;

·       Выполнять только для новых должников – при установке флажка в данном реквизите для новых контрагентов-должников, после присвоения им категории, регламентное действие выполняется один раз в течение одной недели.

 

В табличной части элемента справочника содержится список регламентных действий, выполняемых для выбранной категории объекта скоринга. Количество строк в табличной части формы соответствует количеству действий, которые будут выполняться в рамках данного Регламентного действия. Установка или снятие флажка в графе «Используется» позволяет включить или исключить любую строку таблицы, тем самым, включить или исключить любое действие из выполняемого Регламентного действия.
