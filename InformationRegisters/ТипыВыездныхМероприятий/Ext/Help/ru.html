Справочник "Типы мероприятий"

 

Справочник «Типы мероприятий» предназначен для хранения информации о типах мероприятий, проводимых Пользователями по истребованию задолженности контрагента. Элементы справочника «Типы мероприятий» используются при заполнении документа «Мероприятия», например, из подсистемы «Рабочий стол».

Справочник имеет многоуровневую структуру. Часть элементов справочника является предопределенной, что связано с их обязательным участием в картах бизнес-процессов.

Форма элемента справочника состоит из:

·       рабочей области с реквизитами, содержащими данные о типе мероприятий, и панели «Специальные опции»;

·       области навигации, которая по ссылке позволяет перейти к связанному справочнику «Результаты мероприятий».

 

Особенностями рабочей области элемента справочника являются реквизиты:

·       Функции для дополнительных данных – функция (алгоритм) получения дополнительных данных при проведении мероприятий (выбирается из элементов справочника «Функции дополнительных данных»);

·       Функция перед созданием из БП – возможность создания мероприятия непосредственно из бизнес-процесса (выбирается из элементов справочника «Функции дополнительных реквизитов»).

 

Панель «Специальные опции» содержит следующие реквизиты:

·       Не отображать в планировщике – в случае установки флажка в данном реквизите, мероприятие с данным типом не будет отображаться в Планировщике сотрудников организации (см. подсистему «Рабочий стол»);

·       Кнопка печати шаблона – при установке флажка в данном реквизите Пользователю становится доступной кнопка формирования печатной формы документа «Мероприятия».
