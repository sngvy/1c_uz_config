Справочник "Интерфейсы"

 

Справочник «Интерфейсы» служит для хранения информации об интерфейсах, использующихся в системе различными сотрудниками. Элементы справочника доступны для создания просмотра и редактирования только Администратору системы.

Данный справочник имеет одноуровневую структуру.

 

Табличная часть справочника «Интерфейсы» содержит формы с настройками, которые выбираются Администратором для данного интерфейса из справочника «Формы».
