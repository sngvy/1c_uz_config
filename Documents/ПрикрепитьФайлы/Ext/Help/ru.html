Документ "Прикрепить файлы"

 

Документ «Прикрепить файлы» предназначен для регистрации факта прикрепления файла того или иного расширения к объектам системы. Создание новых документов может осуществляться:

·       из журнала документов «Прикрепить файлы» при помощи стандартных функций управления командной панели;

·      из области навигации элементов справочников «Долговые обязательства», «Исполнительные документы, «Договоры контрагентов» и т.д.

 

Форма документа состоит из реквизитов шапки формы и табличной части. Реквизиты шапки формы документа при создании документа из области навигации элементов справочников заполняются автоматически на основании данных соответствующих элементов справочников. При создании документов из области навигации элементов справочников часть реквизитов заполняется автоматически.


В табличную часть документа «Прикрепить файлы» вносятся данные о типе прикрепляемых файлов к указанному объекту:

·       № – порядковый номер прикрепляемого файла;

·      Тип файла – тип прикрепляемого файла в соответствии с элементами справочника «Типы прикрепляемых файлов»;

·       Путь – путь к директории размещения загружаемого файла;

·       Размер – размер прикрепляемого файла;

·       Загружен – в данном реквизите устанавливается флажок при загрузке файла в систему.
